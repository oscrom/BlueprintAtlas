## Getting Started

### Prerequisites

* Visual Studio 2022 (v17.0)
* Windows 10, version 2004.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/oscrom/BlueprintAtlas.git
   ```
2. Restore Nuget Packages
   ```sh
   nuget restore
   ```
3. Rebuild All

**Note: drag/drop while debugging via Visual Studio does not work. Use Release**
